clip-path

o clip-path é uma propriedade css que permite criar formas complexas cortando elementos de acordo com um caminho especifico. é util para criar layouts e efeitos visuais interessantes.

como funciona o clip-path

oclip-pathdefine uma area visivel de um elemento, escondendo o que está fora dessa área. existem várias formas que você pode usar com clip-path, como:

  • polygon(): define uma forma usando uma lista de coordenadas que formam um poligono.
  • circle(): define uma forma circula.
  • ellipse(): define uma forma elípticas.
  • inset(): define um retângulo interno com uma margem opcional.
